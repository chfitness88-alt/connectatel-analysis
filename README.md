# 📊 Análisis ConnectaTel

Análisis de datos del comportamiento de clientes de telecomunicaciones (llamadas y mensajes) en Latinoamérica.

## 🎯 Objetivo

Identificar patrones de uso, detectar outliers y segmentar clientes para optimizar la oferta comercial.

## 📂 Datasets

- plans.csv → planes disponibles  
- users_latam.csv → datos de clientes  
- usage.csv → uso real (llamadas y mensajes)  

## ⚙️ Proceso

- Limpieza de datos (sentinels, nulos, fechas)
- Análisis exploratorio (EDA)
- Visualización de distribuciones
- Detección de outliers (IQR)
- Segmentación de clientes (edad y uso)

## 📊 Insights clave

- Existe un grupo reducido de usuarios de alto consumo
- Distribución sesgada a la derecha en uso
- Datos con errores de captura (edad, fechas)

## 💡 Recomendaciones

- Crear planes diferenciados por consumo
- Optimizar planes básicos
- Fidelizar usuarios de alto valor

Abrir el notebook `.ipynb` y ejecutar las celdas en orden.
